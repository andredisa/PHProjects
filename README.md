# PHP Projects

This repository contains various projects developed in PHP, ranging from games to useful calculations and everyday applications. Below you will find a description of each project included in this repository.

## Included Projects:

1. **Minesweeper**
   - **Description:** A classic logic and concentration game, Minesweeper allows the user to uncover all the cells on the field without triggering the mines. The game includes different difficulty levels and a scoring system.

2. **Calculator**
   - **Description:** A simple calculator developed in PHP that allows basic mathematical operations such as addition, subtraction, multiplication, and division. The interface is user-friendly, designed to simplify everyday calculations.

3. **Vacation Exercise: CO2 Emissions Calculation**
   - **Description:** An application to calculate CO2 emissions resulting from meat consumption during a vacation. The user enters the type and amount of meat consumed, and the app provides an analysis of the associated greenhouse gas emissions.

4. **Goose Game**
   - **Description:** A traditional game, the Goose Game, developed in PHP. The user rolls a die to advance on the board, facing various obstacles and challenges along the way. The goal is to be the first to reach the end of the game.

5. **SIAE for Music**
   - **Description:** An application to manage the registration of musical works for SIAE (Italian Society of Authors and Publishers). It allows authors to register their works and gain control over their musical compositions.

6. **Slot Machine**
   - **Description:** A Slot Machine game developed in PHP. The user can try their luck with different combinations of symbols, aiming to hit the jackpot. It includes a winning system and random outcome management.

7. **To-Do List**
   - **Description:** A simple To-Do List that allows the user to add, modify, and remove tasks from a list. A useful application for organizing daily commitments and improving productivity.
